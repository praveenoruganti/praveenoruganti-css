Here with the CSS Border properties 

1. border-width: Thickness of border, we can give values in px,cm,mm.
2. border-style: solid, dotted, dashed, double
3. border-color: text, hex, rgb
4. border: width style color
   border-left: width style color
   border-right: width style color
   border-top: width style color
   border-bottom: width style color
5. border-radius: px or %
   border-radius: TL TR BR BL;
